# Reading a user Did Document

# REST API

# Endpoint

/identities/${did}
1

# Parameters

Parameter Description
did Address of the user for which to read the Did Document

# Example

# Call

http://localhost:1317/identities/did:com:15erw8aqttln5semks0vnqjy9yzrygzmjwh7vke
